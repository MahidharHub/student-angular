Student Demo

Welcome, this is a demo application of Angular 5 and Spring Boot (Spring framework). In this demo application ,the following technologies has been used

1. Angular 5
2. Bootstrap 4
3. RX JS(client side search)
4. ngx-Bootstrap(client side pagination)
5. Spring Boot
6. Spring Rest
7. Spring JPA
8. H2 DB
9. Swagger
10. Hystrix
